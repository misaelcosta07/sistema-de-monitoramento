🚨 Mini Service Desk
📘 Sobre o projeto

O Mini Service Desk é uma aplicação web simples que simula um ambiente de monitoramento e suporte técnico (Service Desk / NOC).

O sistema foi desenvolvido com o objetivo de representar, de forma prática, o fluxo básico de gerenciamento de incidentes em uma infraestrutura de TI.

🎯 Objetivo

Simular o dia a dia de um analista de suporte, incluindo:

Monitoramento de serviços
Detecção de falhas
Abertura de chamados
Atualização de status
Registro de eventos (log)
🧠 Funcionalidades
🖥️ Monitoramento
Exibição do status de um servidor (Online / Offline)
Simulação de falha manual
🎫 Chamados
Criação automática de chamados ao detectar falhas
Atualização de status:
Aberto
Em andamento
Resolvido
📞 Log de eventos
Registro em tempo real das ações do sistema
Exibição com horário automático

JavaScript
